GSP_STATIONARITY_RATIO - Assert the stationarity level of some data

Usage

r = gsp_stationarity_ratio(G, C)

Input parameters

G Graph
C Covariance matrix

Output parameters

r Ratio

Description

This function compute the ratio of energy contained into the diagonal of the Fourier covariance matrix:

\begin{equation*} T = U^{*} C U \end{equation*}

References:

N. Perraudin and P. Vandergheynst. Stationary signal processing on graphs. In Infoscience - EPFL, 2016.
